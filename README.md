Geo|DST R&D Template 1
====

# Summary
This is an R&D template utilizing [BIDS](http://bids.state.gov) data.

# Tech Details
* We well use the [OpenGeo suite](http://opengeo.org/), with LeafletJS and Bootstrap on the frontend.
